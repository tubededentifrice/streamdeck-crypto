How alert rules work

Alert rule are JavaScript snippets that will be evaluated upon value change.

When returning true, the text and background colors will be inverted.

value is the displayed value.

Alerts are only supported in "ticker" mode (ie not candlebars).

How color rules work

Color rule are JavaScript snippets that will be evaluated upon value change.

When returning a non empty color, it will override the default ones.

value is the displayed value.

alert is a boolean indicating whether the alert mode is triggered (ie. the "Alert rule").

Color rules are only supported in "ticker" mode (ie not candlebars).
